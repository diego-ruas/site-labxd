# LABxD Conf 2026

Este repositório contém a página web institucional para o evento **LABxD Conf 2026**, organizado pelo Laboratório de Design de Interação (LABxD) da Universidade Federal de Pelotas (UFPel).

O site está hospedado temporariamente (apenas para fins de teste e demonstração) em: [labxd.diegoruas.com.br](http://labxd.diegoruas.com.br)

## Aviso Importante (Uso Educacional)

* **Fins Educacionais:** Este projeto foi desenvolvido e disponibilizado **apenas para propósitos educacionais**.
* **Proibida a Replicação:** **Não é permitida a replicação, redistribuição ou cópia** deste código e layout para outros fins, uma vez que se trata da identidade visual e do design de uma página institucional específica e real.

## Como Visualizar o Projeto

Por se tratar de um site estático construído puramente com HTML e CSS, você pode visualizar a página diretamente no seu navegador:

1. Baixe ou clone o repositório.
2. Navegue até a pasta do projeto.
3. Abra o arquivo `index.html` em qualquer navegador web de sua preferência.

## Tecnologias Utilizadas

* **HTML5** para a marcação estrutural e semântica.
* **CSS3** para estilização, layout responsivo e transições.
